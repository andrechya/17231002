Dalam era digital yang serba cepat, metode pembayaran non-tunai telah menjadi pilihan utama. Aplikasi Kredivo menghadirkan fitur pembayaran QRIS untuk memudahkan pengguna dalam bertransaksi di berbagai merchant tanpa perlu membawa uang tunai. Prosesnya aman, cepat, dan dapat dilakukan hanya dengan beberapa langkah melalui aplikasi Kredivo.

Proses dimulai saat pengguna melakukan pembelian di merchant. Pengguna cukup meminta merchant menampilkan kode QRIS yang berisi detail transaksi. Selanjutnya, pengguna membuka aplikasi Kredivo, memilih menu Bayar dengan QRIS, dan melakukan pemindaian kode QR tersebut. Aplikasi akan menampilkan detail pembayaran dan pilihan tenor cicilan.

Setelah pengguna memilih metode pembayaran dan mengonfirmasi transaksi, sistem Kredivo akan memproses otorisasi pembayaran. Jika berhasil, pengguna menerima notifikasi bahwa transaksi telah selesai dan bukti pembayaran otomatis tersimpan di riwayat transaksi.

Activity Diagram Pembayaran QRIS

Diagram berikut menunjukkan alur aktivitas pembayaran QRIS di aplikasi Kredivo, mulai dari pemindaian kode QR hingga konfirmasi transaksi berhasil. Diagram ini membantu memahami hubungan antara pengguna, aplikasi, dan sistem QRIS secara visual.

  • User memulai transaksi di merchant.
  • Merchant menampilkan kode QRIS.
  • User membuka aplikasi Kredivo dan memilih menu "Bayar dengan QRIS".
  • User memindai kode QR.
  • Aplikasi mengirim data QRIS ke sistem Kredivo untuk verifikasi.
  • Sistem memverifikasi data dan menampilkan detail transaksi.
  • User memasukkan PIN untuk konfirmasi pembayaran.
  • Sistem memproses dan menampilkan notifikasi transaksi sukses.
